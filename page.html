My Projects

Real-Time Signal Viewer

An advanced tool for real-time signal visualization and analysis. It supports multiple signals with features for customization, detailed zooming, and control. Generate comprehensive PDF reports, perform in-depth statistical analysis, and synchronize signals across graphs for integrated insights. Ideal for both research and practical applications, it offers a user-friendly interface for efficient signal management and reporting.

Sampling Studio

A desktop application that illustrates signal sampling and recovery using the Nyquist-Shannon theorem. It enables sampling and reconstruction of analog signals, showing the impact of sampling rates and noise on signal accuracy. Users can work with both pre-recorded and custom signals, and observe real-time effects. Its intuitive design makes it a valuable tool for learning and applying digital signal processing principles.

Signal Equalizer

A Python desktop application for signal processing with four modes: Uniform Range, Musical Instruments, Animal Sounds, and ECG Signals. It allows users to load, visualize, and adjust signals using sliders, analyze them in the Fourier domain, and view detailed plots and spectrograms. With various smoothing options, it offers a streamlined and intuitive approach to mastering signal analysis and manipulation.

FT Image Mixer

A Python desktop app using PyQt5 for advanced image processing. It allows users to convert images to grayscale, resize them uniformly, and view components like Magnitude and Phase. Users can mix components and regions from multiple images, adjust brightness and contrast, and benefit from multithreading for enhanced performance. Results and interactions are logged for easy reference, making it a versatile tool for image manipulation.

Fingerprint Security Voice code Access

This Python desktop application uses voice fingerprint and spectrogram technology to identify up to 8 individuals. It operates in two modes: Security Voice Code, requiring a specific pass-code sentence for access, and Security Voice Fingerprint, granting access based on vocal match. The system calculates probabilities to determine access, ensuring secure authentication.

Realtime Digital Filter Design

This desktop application lets users design digital filters by placing and adjusting zeros and poles on the z-plane. It features interactive z-plane plotting, frequency response visualization, and real-time signal filtering. Users can also apply and customize all-pass filters to correct phase distortions, with options to visualize and control the filtering process.
